# ⛽ fuel-cli-ui

Landing page for [@shan8851/fuel-cli](https://www.npmjs.com/package/@shan8851/fuel-cli) — UK fuel prices in your terminal.

**🌐 [fuel-cli.xyz](https://fuel-cli.xyz)** · **📦 [npm](https://www.npmjs.com/package/@shan8851/fuel-cli)** · **💻 [CLI source](https://github.com/shan8851/fuel-cli)**

## Tech

Vite + React 19 + TypeScript + Tailwind CSS v4 + Motion. Dark theme, terminal examples, static SPA.

## Local dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # output → dist/
```
